OFKITECH HRMS

OfkiTech HRMS is a conseptual project for office employees to fulfil their daily basic HR related needs. OfkiTech HRMS allows employees to mark their attendance, leaves and holidays along with some additional benefits.

Problem Statement

Working day of an employee starts with marking an attendance, and then accepting attendance requests if he/she is a Manager / Reporting Manager. Considering a scenario of an employee being away from base office (Out-Sourced or Working from different location), it's hectic to mark an attendance via email or sheets, And also to track that as well. One of the primary problems of accessibility was to help users access the platform on mobile devices.

Solution and Visual Design

Considering a platform to work as a quick solution to traditional attendance methods, And also do not wanted to engage user in extra/fancy features to waste his/her valuable time, responsive design is selected over adaptive design as design solution. User can utilise features from any location and device.
To fulfil primary need of an employee that is 'Marking an attendace,' we need to create a page with suitable controls for employees. Employees can apply attendace in different format like Swipe, Outdoor Duty etc.
Secondary features like tracking attendace, applying leaves, getting holiday list will have separate pages, notations with approprite controls and values to achieve users goal.
On Dashboard calendar can work as a quick attendance, leave or Outdoor Duty application.
